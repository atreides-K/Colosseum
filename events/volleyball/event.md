## Format

- Pools → League → Knockouts (if needed) → Quarterfinals → Semifinals → 3rd-place match → Final
- Best of 3 sets (25-25-15 points), except Semis/Finals
- League scoring: 2 points for win, 1 for technical forfeit, 0 for loss
- Tiebreak: sets won → point difference

## Rules

- FIVB rules with NO ROTATION (fixed player positions)
- Players verified before each game via Google form
- Guest locked to one department once played; substitution allowed within guest limit
- 10-minute grace period before walkover
- Match schedule: 5 PM – 12 AM, including holiday mornings if needed
- Open to all IISc community (students, staff, professors, alumni, interns)
