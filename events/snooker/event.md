## Format

- Knockout throughout

### Men's Doubles
- Rounds 1–2: 1 singles (30min, senior table) + 1 singles (30min, junior table) + 1 doubles tiebreaker (45min)
- QF: 2 singles (30min each, both tables) + 1 doubles tiebreaker (45min)
- SF: 3 singles each table + 1 doubles tiebreaker
- Finals: 3 singles each table + 3 doubles

### Mixed Doubles
- Rounds 1–2: same as Men's
- QF: 1 singles (45min each table) + 1 doubles tiebreaker (60min)
- SF: 2 singles (45min each) + 1 doubles tiebreaker (60min)
- Finals: 2 singles each table + 3 doubles (60min each)

### Women's Doubles
- Format depends on number of registrations

## Rules

- WPBSA rulebook applies
- Players pre-decide table allocation, cannot switch
- Referee decision final (Free Ball, Touching Ball, disputes)
