# 🎱 Snooker Rules — Spectrum '26

## Categories

1. Open Singles
2. Men's Doubles
3. Mixed Doubles
4. Women's Doubles

All matches follow a **knockout format**. Guest participation follows general Spectrum rules.

## Men's Doubles Format

| Round | Format |
|-------|--------|
| **Rounds 1 & 2** | 1 singles (30 min, senior table) + 1 singles (30 min, junior table) + 1 doubles tiebreaker (45 min) |
| **Quarterfinals** | 2 singles (30 min each, both tables) + 1 doubles tiebreaker (45 min) |
| **Semifinals** | 3 singles on each table + 1 doubles tiebreaker |
| **Finals** | 3 singles on each table + 3 doubles |

## Mixed Doubles Format

| Round | Format |
|-------|--------|
| **Rounds 1 & 2** | Same as Men's Doubles |
| **Quarterfinals** | 1 singles (45 min, each table) + 1 doubles tiebreaker (60 min) |
| **Semifinals** | 2 singles (45 min each, each table) + 1 doubles tiebreaker (60 min) |
| **Finals** | 2 singles on each table + 3 doubles (60 min each) |

## Women's Doubles

Format depends on number of registrations.

## General Rules

- Players must **pre-decide table allocation** and cannot switch during the match
- In cases of **Free Ball, Touching Ball, or disputes**, the referee's decision is final
- For complete rules, refer to the official [WPBSA rulebook](https://wpbsa.com/wp-content/uploads/2198_WPBSA-Rulebook-2024-25.pdf)
