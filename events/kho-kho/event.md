## Format

- Two innings of 14 minutes each (7 min chase + 7 min defend, 5-min break after each)
- Finals: 9 minutes per inning
- 1 point per defender out (chasing team)
- League: 2 points for win, 0 for loss

### Tiebreak
- Sudden death: time to catch first opponent runner (lesser time wins)
- If top 2 tied: both qualify for finals
- If 2nd/3rd or 3rd/4th tied: sudden death match
- If top 3 tied: round-robin sudden death

- Referee decision is final, no arguments entertained
