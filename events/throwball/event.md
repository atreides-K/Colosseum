## Format

- Best of 3 sets, 25 points per set (rally scoring)
- 2-3-2 player position on court

## Rules

- Catch with both hands, return with one hand only
- Service: released from above shoulder level, within 5 seconds, from service zone
- 3-second release time after catching
- No double touches
- No simultaneous catching
- 2 timeouts of 30 seconds per set
- Eligibility: college students only, one team per player
